# :bird:遗传算法（Genetic Algorithm）

###  :arrow_forward:**1.前言**

本项目实现了能够添加各种约束条件（等式约束、不等式约束）下的规划模型遗传算法求解，同时可适用于连续变量、整型变量和01变量。
